
## Introduction

azchat - A simple, yet fully functional GPT3.5/GPT4 Rust CLI client using Azure OpenAI API, reqwest, tokio and serde.


## Run Locally

Clone the project

```bash
  git clone https://github.com/bcarnazzi/azchat.git
```

Go to the project directory

```bash
  cd azchat
```

Export environnement variables

```bash
  export AZURE_OPENAI_ENDPOINT="https://my-gpt-endpoint.openai.azure.com/"
  export AZURE_OPENAI_DEPLOYMENTID="mydeploymentid"
  export AZURE_OPENAI_APIKEY="d33dbeaf"
```

Start the client

```bash
  cargo run
```

Have fun :)
